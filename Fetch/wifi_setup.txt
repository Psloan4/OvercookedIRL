Fetch setup instructions:
1. Plug in ethernet
2. run ssh fetch@qiborg
3. run source wifi_ap_setup.sh

Desktop setup instructions:
1. connect to hotspot (password: fetch1234)
2. run source wifi_fetch_setup.sh
3. youre now connected to fetchs ros network and can run nodes on your machine connected to fetch

SSH to wifi:
1. ssh fetch@10.42.0.1
2. this remotely logs you on to fetchs software

Publishing compressed image:
rosrun image_transport republish raw in:=/head_camera/rgb/image_raw compressed out:=/head_camera/rgb/image_raw

start gmapping:
rosrun gmapping slam_gmapping scan:=/base_scan _tf_delay:=0.1

start move_base:
roslaunch fetch_navigation fetch_nav.launch
